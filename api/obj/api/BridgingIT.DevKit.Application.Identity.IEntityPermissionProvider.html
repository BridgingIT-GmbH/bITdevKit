Interface IEntityPermissionProvider

Namespace
BridgingIT.DevKit.Application.Identity
Assembly
BridgingIT.DevKit.Application.Identity.dll

Defines the contract for a service that provides access to entity permissions. This interface abstracts the storage and retrieval of permissions, allowing for different implementations.
