Remarks

See https://www.quartz-scheduler.net/documentation/quartz-3.x/tutorial/crontriggers.html for cron syntax details. The 6-field format is: [Seconds] [Minutes] [Hours] [Day of Month] [Month] [Day of Week].

AtDateTime(DateTimeOffset)

Sets the cron expression to trigger exactly once at a specific date and time.

public CronExpressionBuilder AtDateTime(DateTimeOffset dateTime)

Parameters

dateTime DateTimeOffset

The DateTimeOffset specifying the exact date and time for the single execution.

Returns

CronExpressionBuilder

The current CronExpressionBuilder instance for fluent chaining.

Examples

// Run once on March 27, 2025, at 14:30:00 (2:30 PM) var dateTime = new DateTimeOffset(2025, 3, 27, 14, 30, 0, TimeSpan.Zero); var cron = new CronExpressionBuilder() .AtDateTime(dateTime) .Build(); // "0 30 14 27 3 ?"

Remarks

This creates a cron expression that matches only the specified date and time within the year. For true one-time execution, consider using a Quartz SimpleTrigger instead of a cron trigger.

Build()

Builds the final 6-field cron expression string.

public string Build()

Returns

string

The constructed cron expression as a string.

Examples

// Build a cron for a specific date and time: 15th of the month, Wednesday, at 12:59 PM var cron = new CronExpressionBuilder() .DayOfMonth(15) .DayOfWeek(CronDayOfWeek.Wednesday) .Hours(12) .Minutes(59) .Build(); // "0 59 12 ? * WED"
