CompressAsync(IFileStorageProvider, string, string, IProgress<FileProgress>, FileCompressionOptions, CancellationToken)

Writes a compressed file to the storage provider by compressing a file or all files and subdirectories in a directory using SharpCompress.

public static Task<Result> CompressAsync(this IFileStorageProvider provider, string path, string inputPath, IProgress<FileProgress> progress = null, FileCompressionOptions options = null, CancellationToken cancellationToken = default)

Parameters

provider IFileStorageProvider

The file storage provider to use for reading the input and writing the compressed file.

path string

The path where the compressed file will be written (e.g., "output.zip").

inputPath string

The path of the file or directory to compress (e.g., "file.txt" or "directory").

progress IProgress<FileProgress>

An optional progress reporter for tracking the compression process.

options FileCompressionOptions

Optional configuration settings for compression. If null, default settings are used.

cancellationToken CancellationToken

A token to cancel the operation.

Returns

Task<Result>

A Result indicating success or failure of the compression operation.

Remarks

This method uses SharpCompress to create a compressed file from a file or directory path in the storage provider, maintaining compatibility with popular tools (e.g., 7-Zip, WinZip). Supported formats: Zip, Tar, GZip. Password protection is not supported during compression with SharpCompress.

ListCompressedFilesAsync(IFileStorageProvider, string, string, IProgress<FileProgress>, FileCompressionOptions, CancellationToken)

Lists all files within a compressed archive stored in the storage provider, optionally handling password-protected archives using SharpCompress.

Parameters

provider IFileStorageProvider

The file storage provider to use for reading the compressed archive.

path string

The path of the compressed archive to read (e.g., "archive.zip").

password string

An optional password for decrypting the compressed archive. If null or empty, no decryption is applied.

progress IProgress<FileProgress>

An optional progress reporter for tracking the listing process.

options FileCompressionOptions

Optional configuration settings for archive handling. If null, default settings are used.

cancellationToken CancellationToken

A token to cancel the operation.

Returns

Task<Result<IEnumerable<string>>>

A Result<IEnumerable<string>> containing the list of file names on success, or an error on failure.

Remarks

This method uses SharpCompress to read archive entries compatible with popular tools (e.g., 7-Zip, WinZip). Supported formats: Zip, Tar, GZip. Returns all file names at once without pagination. Only file entries are included (directories are excluded). Uses DecryptionError if password is wrong or decryption fails.

ReadCompressedFile(IFileStorageProvider, string, string, IProgress<FileProgress>, FileCompressionOptions, CancellationToken)

Reads a compressed file from the storage provider and decompresses it into a stream, optionally handling password-protected compression using SharpCompress.

public static Task<Result<Stream>> ReadCompressedFile(this IFileStorageProvider provider, string path, string password = null, IProgress<FileProgress> progress = null, FileCompressionOptions options = null, CancellationToken cancellationToken = default)

Parameters

provider IFileStorageProvider

The file storage provider to use for reading the compressed file.

path string

The path of the compressed file to read (e.g., "input.zip").

password string

An optional password for decrypting the compressed file. If null or empty, no decryption is applied.

progress IProgress<FileProgress>

An optional progress reporter for tracking the decompression process.

options FileCompressionOptions

Optional configuration settings for decompression. If null, default settings are used.

cancellationToken CancellationToken

A token to cancel the operation.

Returns

Task<Result<Stream>>

A Result<T> containing the decompressed stream on success, or an error on failure.

Remarks

This method uses SharpCompress to read compressed files with password protection compatible with popular tools (e.g., 7-Zip, WinZip). Supported formats: Zip, Tar, GZip. It supports Deflate64 decompression and traditional PKZIP or AES encryption, depending on the original file's configuration. Uses DecryptionError if password is wrong or decryption fails.

Task<Result<IReadOnlyDictionary<string, MemoryStream>>>

A Result<IReadOnlyDictionary<string, MemoryStream>> containing the dictionary on success, or an error on failure.
The dictionary maps archive entry paths (including names) to their decompressed System.IO.MemoryStreams.

Remarks

This method does NOT write to storage, it only returns the contents in-memory as a readonly dictionary. The caller is responsible for disposing the returned MemoryStreams. Uses DecryptionError if password is wrong or decryption fails.
