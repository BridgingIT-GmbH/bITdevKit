DeepCopyAsync(IFileStorageProvider, string, string, bool, string, IProgress<FileProgress>, CancellationToken)

Deep copies a file or directory structure from the source path to the destination path within the same storage location.

public static Task<Result> DeepCopyAsync(this IFileStorageProvider provider, string sourcePath, string destinationPath, bool skipFiles = false, string searchPattern = null, IProgress<FileProgress> progress = null, CancellationToken cancellationToken = default)

Parameters

provider IFileStorageProvider

The file storage provider to use for copying the structure.

sourcePath string

The source path of the file or directory to copy (e.g., "source/folder").

destinationPath string

The destination path where the structure will be copied (e.g., "dest/folder").

skipFiles bool

If true, skips copying files and only copies the directory structure. Default is false.

searchPattern string

An optional search pattern to filter files to copy (e.g., "*.txt"). If null, all files are copied unless skipFiles is true.

progress IProgress<FileProgress>

An optional progress reporter for tracking the copying process.

cancellationToken CancellationToken

A token to cancel the operation.

Returns

Task<Result>

A Result indicating success or failure of the deep copy operation.

Remarks

This method recursively copies the directory structure, including empty directories, and files that match the search pattern (unless skipped) from the source to the destination. It uses the provider's methods to list directories, list files, create directories, and copy files, ensuring thread safety and progress reporting.
