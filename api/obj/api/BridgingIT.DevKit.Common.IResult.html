IsFailure

Indicates whether the result of an operation represents a failure.

bool IsFailure { get; }

Property Value

bool

Remarks

This property returns the inverse of the IsSuccess property. If IsSuccess is false, this property will return true, indicating a failure.

IsSuccess

Indicates whether the operation represented by the result was successful. If true, the operation completed successfully. If false, the operation encountered one or more errors.
