Provides a base class for building specific types of options with support for logging.

public abstract class OptionsBuilderBase<TOption, TBuilder> : OptionsBuilder<TOption>, IOptionsBuilder<TOption>, IOptionsBuilder where TOption : OptionsBase, new() where TBuilder : OptionsBuilderBase<TOption, TBuilder>

Type Parameters

TOption

The type of options to be built, derived from OptionsBase.

TBuilder

The type of the builder itself, allowing for a fluent interface by returning the concrete builder type from methods.
