Class PropertyBag

Namespace
BridgingIT.DevKit.Common
Assembly
BridgingIT.DevKit.Common.Abstractions.dll

PropertyBag is a type-safe dictionary abstraction for key-value pairs, with support for type conversion, change notification, and basic merging/cloning. Strongly-typed keys are supported optionally via PropertyBagKey<T>.
