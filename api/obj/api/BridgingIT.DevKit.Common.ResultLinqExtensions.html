BindAsync<T, TVal>(ICollection<Result<TVal>>, Func<TVal, Task<Result<T>>>)

Binds each Result in a collection to an async function that returns a Result. Preserves messages and errors from previous Results in the chain.

public static Task<ICollection<Result<T>>> BindAsync<T, TVal>(this ICollection<Result<TVal>> results, Func<TVal, Task<Result<T>>> binder)

Parameters

results ICollection<Result<TVal>>

The collection of Results to bind.

binder Func<TVal, Task<Result<T>>>

The async function to apply to each successful Result's value.

Returns

Task<ICollection<Result<T>>>

A task containing a collection of Results from each operation.

Type Parameters

T

The type of the result value.

TVal

The type of items in the input Results.

Examples

var results = await customers
    .ToResult()
    .BindAsync(async customer => await ValidateCustomerAsync(customer))
    .BindAsync(async customer => await SaveCustomerAsync(customer))
    .FlattenAsync();
