IsInRelativeRange(TimeOnly, TimeOnly, TimeUnit, int, DateTimeDirection, bool)

Determines whether a time is inside a relative same-day range around an explicit reference time.

public static bool IsInRelativeRange(this TimeOnly time, TimeOnly reference, TimeUnit unit, int amount, DateTimeDirection direction, bool inclusive = true)

Parameters

time TimeOnly

The time to evaluate.

reference TimeOnly

The same-day reference time.

unit TimeUnit

The relative unit.

amount int

The amount of units.

direction DateTimeDirection

The direction from the reference time.

inclusive bool

Whether the boundaries are included.

Returns

bool

true when the time is inside the range; otherwise false.

Remarks

This method uses same-day semantics and does not infer midnight crossing.
