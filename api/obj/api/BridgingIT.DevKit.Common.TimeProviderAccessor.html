Provides ambient access to the current System.TimeProvider instance. This enables domain entities and other non-DI-aware code to access time without requiring constructor injection or method parameters.

public static class TimeProviderAccessor
Inheritance
object
TimeProviderAccessor

Remarks

The current System.TimeProvider is stored in an System.Threading.AsyncLocal<T> to ensure correct flow across asynchronous operations. In production, it defaults to System.TimeProvider.System. In tests, it can be overridden using TimeProviderAccessor.Current = fake.
