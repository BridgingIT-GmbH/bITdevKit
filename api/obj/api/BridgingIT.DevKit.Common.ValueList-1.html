Struct ValueList<T>

Namespace
BridgingIT.DevKit.Common
Assembly
BridgingIT.DevKit.Common.Abstractions.dll

A memory-efficient collection optimized for very small sets of items (0-2). This struct provides value semantics and avoids heap allocations for the most common scenarios where only a few items need to be stored. It uses a hybrid approach:

  • 0-2 items: Stored directly in the struct (stack/inline)
  • 3+ items: Stored in a List{T} on the heap
