A high-performance stopwatch intended for measuring short time intervals. The ValueStopwatch is a lightweight, value-type alternative to the System.Diagnostics.Stopwatch class.

Remarks

This struct provides high-resolution timing using the underlying system's performance counter. It maintains state in a readonly struct, minimizing heap allocations.
