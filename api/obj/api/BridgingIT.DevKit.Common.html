ResultJsonConverter

A custom JSON converter for Result that writes a compact and client-safe JSON representation.

Behavior: - Writes the following top-level properties: - isSuccess (boolean) - messages (array of strings) - errors (array of error objects) - For general errors, delegates to System.Text.Json.JsonSerializer using the error's type and the provided options. - For ExceptionError, emits a safe, flattened object and deliberately excludes the raw System.Exception instance to avoid unsupported types (e.g., MethodBase) and oversized/PII payloads.

Notes: - Deserialization is not supported and will throw System.NotSupportedException. - This converter is intended for outbound API responses or logging, not for round-tripping back into domain models.

return Rule
    // Basic validation using ValueRules
    .Add(Rules.IsNotEmpty(product.Name))
    .Add(Rules.StringLength(product.Name, 3, 100))
    .Add(Rules.NumericRange(product.Price, 0.01m, 999.99m))

    // Conditional validation with When
    .When(!product.IsDigital, builder => builder
        .Add(Rules.IsNotEmpty(product.ShippingAddress))
        .Add(Rules.StringLength(product.ShippingAddress, 10, 200)))

    // Using Unless (inverse of When)
    .Unless(product.IsDigital,
        Rules.GreaterThan(product.Price, 10m))

    // Combining multiple conditions
    .WhenAll(new[]
    {
        product.Price > 100,
        product.IsDigital,
        product.Categories?.Count > 2
    }, Rules.IsNotEmpty(product.ShippingAddress))
    .Check();
