Class ActiveEntityQueryBase<TEntity, TId>

Namespace
BridgingIT.DevKit.Domain
Assembly
BridgingIT.DevKit.Domain.dll

Provides a fluent DSL for building queries against an ActiveEntity entity. This DSL wraps the underlying ActiveEntity static methods (FindOne, FindAll, Exists, Count, ProjectAll, etc.) and ensures all results are returned as Result<T> or ResultPaged<T>.

public abstract class ActiveEntityQueryBase<TEntity, TId> where TEntity : ActiveEntity<TEntity, TId>
