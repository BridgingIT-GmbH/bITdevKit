Property Value

Remarks

The Errors property provides access to the collection of errors that occurred while applying one or more policies. This list can be used to understand what went wrong and to inspect individual IResultError instances for more details.

IsFailure

Indicates whether the policy result is a failure.

public bool IsFailure { get; }

Property Value

bool

Remarks

The IsFailure property returns true if the policy execution did not succeed. This can be utilized to check the overall status of the applied policies, particularly when determining whether subsequent policy executions or actions should proceed.

IsSuccess

Indicates whether the domain policy execution was successful.

public bool IsSuccess { get; }

Property Value

bool

Remarks

The IsSuccess property returns true if all the applied policies executed without errors; otherwise, it returns false. This property is useful to determine the overall success of the policy execution process.
