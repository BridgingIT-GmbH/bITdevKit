Examples

Example usage through the provider:

var result = await provider.UpdateAsync(
    new Specification<User>(u => u.LastLogin < DateTime.UtcNow.AddYears(-1)),
    set => set
        .Set(u => u.IsActive, _ => false)            // constant
        .Set(u => u.LoginCount, u => u.LoginCount+1) // computed
);

Remarks

This class is used internally by the ActiveEntityInMemoryProvider<TEntity, TId> to simulate bulk update operations in memory.
