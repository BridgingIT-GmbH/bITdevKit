ActiveEntityContext<TEntity, TId>

Represents a context for Active Entity operations, bundling the provider and behaviors resolved from the same DI scope. This ensures that the provider and behaviors share the same lifetime and dependencies (e.g., DbContext).

Lifetime Warning: This context is only valid within the DI scope it was created from. Do not cache or pass it across async boundaries outside of that scope, as it may lead to System.ObjectDisposedException if the underlying scope (and its services) are disposed. Always use it immediately within the creating method (e.g., inside UseAsync<TEntity, TId, TResult>(ActiveEntityContext<TEntity, TId>, Func<ActiveEntityContext<TEntity, TId>, Task<TResult>>) or WithTransactionAsync<TEntity, TId>).

If you need to extend the context with more dependencies, resolve them at creation time and add them as properties here, rather than resolving them later to avoid scope mismatches.
