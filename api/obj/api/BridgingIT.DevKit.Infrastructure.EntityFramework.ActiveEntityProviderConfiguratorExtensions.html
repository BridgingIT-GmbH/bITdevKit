UseEntityFrameworkProvider<TEntity, TId>(ActiveEntityConfigurator<TEntity, TId>, Action<ActiveEntityFrameworkProviderBuilder<TEntity, TId>>)

Configures the Entity Framework provider for the entity using a builder to specify the DbContext type. The DbContext must be pre-registered in the service collection (e.g., via AddDbContext).

public static ActiveEntityConfigurator<TEntity, TId> UseEntityFrameworkProvider<TEntity, TId>(this ActiveEntityConfigurator<TEntity, TId> configurator, Action<ActiveEntityFrameworkProviderBuilder<TEntity, TId>> builder) where TEntity : class, IEntity

Parameters

configurator ActiveEntityConfigurator<TEntity, TId>

The entity configurator.

builder Action<ActiveEntityFrameworkProviderBuilder<TEntity, TId>>

A configuration action for the Entity Framework provider builder.

Returns

ActiveEntityConfigurator<TEntity, TId>

The configurator instance for fluent chaining.

Type Parameters

TEntity

The entity type, inferred from the configurator.

TId

The type of the entity's identifier, inferred from the configurator.

Examples

services.AddDbContext<ActiveRecordDbContext>(options => options.UseSqlServer(connectionString));
services.AddActiveRecord(cfg =>
{
    cfg.For<Customer, CustomerId>()
        .UseEntityFrameworkProvider(o => o.UseContext<ActiveRecordDbContext>())
        .AddLoggingBehavior<Customer, CustomerId>();
});
