Remarks

Derived classes implement Map(IEndpointRouteBuilder) to declare their routes. MapGroup(IEndpointRouteBuilder, EndpointsOptionsBase) creates a configured route group from EndpointsOptionsBase and applies shared endpoint metadata such as OpenAPI tags, authorization, CORS, rate limiting, route name prefix metadata, and custom metadata. All configuration is applied before the derived endpoint class maps individual routes into the returned group.

Example:

public sealed class OrdersEndpoints(OrdersEndpointsOptions options) : EndpointsBase
{
    public override void Map(IEndpointRouteBuilder app)
    {
        var group = MapGroup(app, options);

        group.MapGet("", () => Results.Ok())
            .WithName(options, "List");
    }
}

BuildRouteName(EndpointsOptionsBase, string)

Builds an endpoint route name using the optional prefix configured on the endpoint options.

public static string BuildRouteName(EndpointsOptionsBase options, string name)

Parameters

options EndpointsOptionsBase

The endpoint options that may contain a route name prefix.

name string

The endpoint-specific route name.

Returns

string

The unmodified name when no prefix is configured; the prefix when name is empty; otherwise the prefix and name joined by a single period.

Remarks

ASP.NET Core does not automatically prepend endpoint names from route group metadata. Derived endpoint classes should call this helper before passing names to WithName. The helper trims trailing periods from the prefix and leading periods from the endpoint-specific name.

Example:

options.RouteNamePrefix = "Orders";
var name = BuildRouteName(options, "List"); // Orders.List

ArgumentNullException

MapGroup(IEndpointRouteBuilder, EndpointsOptionsBase)

Creates a route group and applies the shared endpoint group configuration from the supplied options.

public RouteGroupBuilder MapGroup(IEndpointRouteBuilder app, EndpointsOptionsBase options)

Parameters

app IEndpointRouteBuilder

The endpoint route builder that owns the new group.

options EndpointsOptionsBase

The endpoint group options to apply.

Returns

RouteGroupBuilder

A configured route group builder that callers can use to map individual endpoints.

Remarks

The method validates app and options before mapping. It creates the group with GroupPath, optionally normalizes the path, applies tags, OpenAPI group metadata, description visibility, authorization or anonymous metadata, CORS metadata, rate limiting metadata, route name prefix metadata, and custom metadata. Authorization role entries and authentication schemes that are blank are ignored. Anonymous access takes precedence over authorization metadata.

Example:

var group = MapGroup(app, options);
group.MapGet("items", GetItems)
    .WithName(BuildRouteName(options, "GetItems"));

Exceptions

ArgumentNullException

Thrown when app or options is null.
