Remarks

The options describe how an endpoint group is created and which metadata is applied to it. The group configuration is additive and optional: unset values are ignored, while configured values are applied to the generated RouteGroupBuilder. Authorization is applied only when RequireAuthorization is enabled, unless AllowAnonymous is set, in which case anonymous metadata is added and authorization metadata is skipped.

Example:

public sealed class OrdersEndpointsOptions : EndpointsOptionsBase
{
    public OrdersEndpointsOptions()
    {
        GroupPath = "/api/orders";
        GroupTag = "Orders";
        RequireAuthorization = true;
        RequirePolicy = "Orders.Read";
    }
}
