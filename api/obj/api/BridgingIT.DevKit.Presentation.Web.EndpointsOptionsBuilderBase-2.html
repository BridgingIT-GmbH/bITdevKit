Remarks

Concrete endpoint option builders inherit this type to expose shared group configuration without duplicating methods. Each method mutates the underlying Target instance and returns the concrete builder so module-specific builder methods can be chained with the shared methods.

Example:

var options = new OrdersEndpointsOptionsBuilder()
    .GroupPath("/api/orders")
    .GroupTag("Orders")
    .RequirePolicy("Orders.Read")
    .Build();
