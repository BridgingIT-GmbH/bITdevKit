AddHandler<THandler>(bool, int)

Adds an additional exception handler type conditionally with optional priority. The handler is only registered if the condition is true.

public GlobalExceptionHandlerOptions AddHandler<THandler>(bool when, int priority = 0) where THandler : IExceptionHandler

Parameters

when bool

The condition that must be true for the handler to be registered.

priority int

The priority of the handler. Higher values are executed first. Default is 0.

Returns

GlobalExceptionHandlerOptions

The current GlobalExceptionHandlerOptions instance for chaining.

Type Parameters

THandler

The type of the exception handler to add.

Examples

options.AddHandler<DebugExceptionHandler>(
           when: environment.IsDevelopment(),
           priority: 100)
       .AddHandler<ProductionExceptionHandler>(
           when: environment.IsProduction());
