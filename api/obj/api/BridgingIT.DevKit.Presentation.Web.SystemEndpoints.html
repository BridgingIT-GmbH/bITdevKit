Remarks

The endpoint set maps routes under SystemEndpointsOptions.GroupPath when the options are enabled. Individual echo, info, and modules routes are controlled by their matching option flags. The info endpoint can hide machine, process, network, and runtime details when sensitive information should not be exposed.

Example:

services.AddEndpoints(new SystemEndpoints(new SystemEndpointsOptions
{
    InfoEnabled = true,
    HideSensitiveInformation = true
}));

GetInfo(HttpContext, CancellationToken)

Returns runtime, request, memory, configuration, custom metadata, and uptime information for the application.

public Task<IResult> GetInfo(HttpContext httpContext, CancellationToken cancellationToken)

Parameters

httpContext HttpContext

The current HTTP context used to determine request-local information.

cancellationToken CancellationToken

A cancellation token used while resolving host addresses.

Returns

Task<IResult>

An HTTP 200 result containing SystemInfo on success; otherwise an HTTP 500 problem result when system information cannot be collected.

Remarks

The method collects process, runtime, operating system, memory, configuration, and uptime values. When HideSensitiveInformation is enabled, machine, network, process, runtime, memory, and configuration details are returned as empty strings while custom metadata and uptime are still included. DNS lookup failures for host addresses are logged as warnings and represented as N/A. Unexpected collection failures are logged as errors and returned as problem details.

Returns

IResult

An HTTP 200 result containing module name, priority, enabled state, and registration state; otherwise an HTTP 500 problem result when module projection fails.

Remarks

The response projects each supplied module into a new SystemModule instance and does not expose additional module implementation details. Exceptions are logged and converted to problem details.

Remarks

The method returns without mapping routes when Enabled is false. The root system route is always mapped when enabled; echo, info, and modules are mapped only when their corresponding option flags are enabled. Routes inherit the group configuration applied by MapGroup(IEndpointRouteBuilder, EndpointsOptionsBase).
