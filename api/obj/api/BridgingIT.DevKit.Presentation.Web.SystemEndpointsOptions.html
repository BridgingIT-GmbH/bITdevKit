Remarks

The default group path is /_bdk/api/system and the default tag is _bdk.System. The root system route is mapped whenever the endpoint set is enabled, while the echo, info, and modules routes can be enabled or disabled independently.

Example:

var options = new SystemEndpointsOptions
{
    InfoEnabled = true,
    ModulesEnabled = false,
    HideSensitiveInformation = true
};

Properties

CustomMetadata

Gets or sets custom metadata included in the system info response.

public Dictionary<string, string> CustomMetadata { get; set; }

Property Value

Dictionary<string, string>

Remarks

The dictionary is returned as CustomMetadata by GetInfo(HttpContext, CancellationToken). It can be used for non-sensitive application or deployment information that should be exposed through the system endpoint.

EchoEnabled

Gets or sets whether the echo route is mapped.

public bool EchoEnabled { get; set; }

Property Value

bool

Remarks

When enabled, GetSystem(HttpContext) includes an echo link and Map(IEndpointRouteBuilder) maps the echo route. The default value is true.

HideSensitiveInformation

Gets or sets whether sensitive runtime and host details are hidden from the info response.

public bool HideSensitiveInformation { get; set; }

Property Value

bool

Remarks

When enabled, GetInfo(HttpContext, CancellationToken) replaces host, IP, process, framework, machine, operating system, memory, URL, and timezone values with empty strings. Custom metadata and uptime remain visible.
